# react-native-loader-kit

✳️Purely native loader indicators for React Native✳️

## Installation

```sh
npm install react-native-loader-kit
```

## Usage

```js
import { LoaderKitView } from "react-native-loader-kit";

// ...

<LoaderKitView color="tomato" />
```
